# OctoPrint_IFTTTmaker

Connect Octoprint to the IFTTT maker channel.

## Setup

Install via the bundled [Plugin Manager](https://github.com/foosel/OctoPrint/wiki/Plugin:-Plugin-Manager)
or manually using this URL:

    https://github.com/sedgett/OctoPrint_IFTTTmaker/archive/master.zip

## Configuration

1. Setup the IFTTT maker channel at https://ifttt.com/maker/ and enter your key in the settings screen.
2. Create recipies on IFTTT
3. Select the events to send to IFTTT via settings.
